Description

AstaRCMS is a multi-purpose CMS (Content Management System). It can be used for blogs, portals, company and agency websites, magazines, newspapers and many other. It is clean, simple, lightweight, responsive and user-friendly.

Features

  • Posts: Posts can be added, edited and deleted. WYSIWYG HTML editor. Users can also added & edited post after registration. But can't be delete posts. User post will be published, if admin approved this post.
  • Categories: Only admin can create categories. Posts can be separated to categories. Every categories will have different permalink. Permalink will be generated automatically. Suppose, you have created a category named "Android Apps Review". The permalink for "Android Apps Review" will be "android-apps-review" and the link of the category will be "https://yoursitename.com/category/android-apps-review". You can change the permalink later.
  • Comments: Users & guest can share their opinion in the comments section of the posts. Registered users comments will be published instantly. But guests comments will be published after admin approvel. Duplicate comments detection & captcha verification feature.
  • Users Registration: User-friendly login, registration page and dashboard.
  • Search Module: Searches for posts and displays results as a list in seach page.
  • Pages: HTML supported About Us, Contact Us, Advertise & Privacy Policy page. Built-in contact us form in contact us page with captcha. You can change page status from active to inactive and inactive to active.
  • Responsive Design: Looks good on many devices and screen resolutions.
  • Advertisement: Advertisement module allows you to monetize your site. You can put advertisement in different place.
  • Seo Friendly Url: Seo friendly posts, page, categories and pagination url.
  • Admin Panel: The whole website can be managed via the Admin Panel. Admin can create/edit/delete post, category, media, activate or inactivate user, change user password and many more.
  • Site Settings: Dynamically update site logo, favicon, title, subtitle, keywords, social link etc.
  • Post Per Page: Admin can set how many post can be displayed in home page, search page & category page.
  • Theme: Many themes are added by default.
  • Post View Count: Display post view counter in post page and admin dashboard.
  • XML sitemap: Posts link will added automatically in sitemap.xml file. Sitemap link: https://yoursitename.com/sitemap.xml

and many more...

Requirements

  • PHP 5.3+
  • MySQLi

Installation & Configuration

  • Import the mysql.sql file from the "MySQL" folder into your MySQL database using the import function available in MySQL.
  • With a text editor open the following file located inside the "Script/include/" folder: config.php, and replace what's between the quotes (e.g: 'YOURDBNAME') with your MySQL settings:

    • $hostname = "YOURDBHOST";
    $db_username = "YOURDBUSER";
    $db_password = "YOURDBPASS";
    $db_name = "YOURDBNAME";

  • Upload the files from the "Script" folder on the FTP server (usually public_html folder when installing it on a domain).
  • Now, to access your Admin Panel go to https://yoursitelink.com/admin and login using the default credentials:

    • Email: admin@admin.com

    Password: admin

  • To change admin email & password go to the "Admin Settings" from Dashboard.
  • To change site information like Site Title, Description, Keywords, Post Per Page, Logo, Favicon, Theme and Social Links go to the "Site Settings".
